How the Longest Chain Rule Works:

This demo shows how blockchain forks occur and are resolved. The process works as follows:

1. Normal Mining: New blocks appear in the shared area above the fork point
2. Fork Creation: When a conflict occurs, the last block splits into both chains
3. Competing Chains: The chains below the fork point compete for validity
4. Longest Chain Rule: The chain with more blocks becomes the canonical blockchain
5. Resolution: Use "Resolve Conflict" to see how the longest chain wins

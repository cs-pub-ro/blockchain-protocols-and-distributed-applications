üìö How Consensus Groups Work

Consensus groups are collections of validators that work together to validate transactions and create new blocks. Unlike individual validators, consensus groups must reach agreement before any action is taken.

  • Proposal Phase: A validator proposes a new block or transaction
  • Voting Phase: All validators in the group vote on the proposal
  • Consensus Threshold: A minimum percentage of validators must agree (typically 2/3)
  • Finalization: Once consensus is reached, the block is finalized and added to the blockchain
  • Conflict Resolution: If validators disagree, the group must resolve conflicts through additional voting rounds

Key Benefits: Higher security through collective validation, resistance to individual validator failures, and democratic decision-making process.

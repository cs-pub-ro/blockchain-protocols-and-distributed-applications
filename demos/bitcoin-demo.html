How it works:

1. Transactions: Users create transactions that move Bitcoin from one address to another
2. Mining: Miners compete to solve a cryptographic puzzle (Proof of Work)
3. Block Creation: The first miner to solve the puzzle creates a new block
4. Blockchain: Blocks are chained together, creating an immutable ledger
